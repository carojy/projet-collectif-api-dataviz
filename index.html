Explications

Le but de ce projet est de créer une page web affichant de manière visuelle et dynamique les données de l'API du gouvernement Hub'Eau. Les données de cette API issues des relevés piézométriques reflètent le niveau des nappes phréatiques. La donnée choisie pour le projet concerne l'évolution d'une nappe phréatique du sud de la France à 10 ans d'écart.

Scope: du 20 Janvier 2014 au 2024 (hiver), ville de Rochefort-du-Gard (30), code BSS 09405X0229/S3.
